## 系统概述

YIR Approval Hub 是一个企业级审批流程管理系统，支持员工提交各类申请并由审批人进行审批。

### 用户角色

系统支持三种角色：

1. **员工 (EMPLOYEE)**: 
   - 可以创建和提交申请
   - 查看自己的申请状态
   - 撤回待审批的申请

2. **审批人 (APPROVER)**:
   - 拥有员工的所有权限
   - 可以审批其他人的申请
   - 查看待审批任务列表

3. **管理员 (ADMIN)**:
   - 拥有系统最高权限
   - 可以管理用户、部门和岗位
   - 查看所有申请和审批记录

### 申请类型

系统当前支持两种申请类型：

1. **请假申请**: 病假、年假、事假、产假、陪产假等
2. **报销申请**: 差旅、招待、办公用品、培训、通讯等

## 员工操作指南

### 1. 创建请假申请

#### 步骤：
1. 点击左侧菜单"新建请假申请"
2. 填写申请表单：
   - **标题**: 申请的简短描述（如"春节假期申请"）
   - **请假类型**: 选择请假类型（病假、年假等）
   - **开始日期**: 选择请假开始日期
   - **结束日期**: 选择请假结束日期
   - **请假天数**: 输入请假天数
   - **请假原因**: 详细说明请假原因

3. 选择操作：
   - **保存草稿**: 保存为草稿状态，可以稍后编辑和提交
   - **保存并提交**: 直接提交到审批流程
   - **取消**: 取消本次操作

#### 示例：

```
标题: 年假申请
请假类型: 年假
开始日期: 2024-02-01
结束日期: 2024-02-05
请假天数: 5
请假原因: 春节期间回家探亲
```

### 2. 创建报销申请

#### 步骤：
1. 点击左侧菜单"新建报销申请"
2. 填写申请表单：
   - **标题**: 报销申请的简短描述
   - **报销类型**: 选择报销类型（差旅、招待等）
   - **报销金额**: 输入报销金额（单位：元）
   - **费用日期**: 选择费用发生日期
   - **报销用途**: 详细说明报销用途

3. 选择操作：
   - **保存草稿**: 保存为草稿
   - **保存并提交**: 直接提交审批
   - **取消**: 取消操作

#### 示例：

```
标题: 北京出差报销
报销类型: 差旅
报销金额: 1580.50
费用日期: 2024-01-15
报销用途: 参加北京总部培训，包括往返机票、住宿费用
```

### 3. 查看我的申请

#### 步骤：
1. 点击左侧菜单"我的申请"
2. 查看申请列表，包括：
   - 申请编号
   - 申请类型
   - 标题
   - 状态（草稿、审批中、已通过、已拒绝、已撤回）
   - 创建时间

3. 可以进行的操作：
   - **查看**: 查看申请详情
   - **提交**: 提交草稿状态的申请
   - **撤回**: 撤回审批中的申请

### 4. 查看申请详情

#### 步骤：
1. 在申请列表中点击"查看"按钮
2. 查看申请的完整信息：
   - 基本信息（编号、状态、申请人等）
   - 申请内容详情
   - 审批流程（时间线展示）
   - 审批人意见

#### 审批状态说明：

- 🔵 **草稿**: 申请已创建但未提交
- 🟡 **审批中**: 申请已提交，等待审批
- 🟢 **已通过**: 所有审批人都已同意
- 🔴 **已拒绝**: 某个审批人拒绝了申请
- ⚪ **已撤回**: 申请人主动撤回

### 5. 提交申请

#### 步骤：
1. 在"我的申请"列表中找到草稿状态的申请
2. 点击"提交"按钮
3. 确认提交
4. 申请状态变更为"审批中"

**注意**: 只有草稿状态的申请才能提交！

### 6. 撤回申请

#### 步骤：
1. 在"我的申请"列表中找到审批中的申请
2. 点击"撤回"按钮
3. 确认撤回
4. 申请状态变更为"已撤回"

**注意**: 
- 只有审批中的申请才能撤回
- 撤回后的申请不能重新提交，需要创建新申请

## 审批人操作指南

审批人除了拥有员工的所有权限外，还可以审批其他人的申请。

### 1. 查看待审批任务

#### 步骤：
1. 点击左侧菜单"待审批任务"
2. 查看待处理的审批任务列表，包括：
   - 申请编号
   - 申请类型
   - 标题
   - 审批步骤
   - 申请人
   - 创建时间

### 2. 审批申请

#### 步骤：

**方式一：在待审批任务列表直接审批**
1. 在待审批任务列表中找到要处理的任务
2. 点击"同意"或"拒绝"按钮
3. 输入审批意见（可选）
4. 点击"确定"完成审批

**方式二：查看详情后审批**
1. 点击"查看详情"按钮
2. 仔细查看申请的完整信息
3. 点击浏览器返回按钮回到待审批列表
4. 进行审批操作

#### 审批意见示例：

**同意的意见：**
- "同意该申请"
- "已核实，准予批准"
- "符合规定，同意"

**拒绝的意见：**
- "请假时间与重要项目时间冲突，建议调整"
- "报销金额超出预算，请重新申请"
- "资料不全，请补充后重新提交"

### 3. 查看审批历史

在申请详情页面可以查看完整的审批历史，包括：
- 创建时间
- 提交时间
- 每个审批节点的处理时间
- 审批人意见
- 最终结果

## 常见问题

### Q1: 忘记密码怎么办？

**A**: 请联系系统管理员重置密码。管理员可以直接在数据库中修改密码。

### Q2: 可以修改已提交的申请吗？

**A**: 不可以。已提交的申请无法修改。如需修改，请先撤回申请，然后创建新的申请。

### Q3: 草稿会自动保存吗？

**A**: 不会。需要手动点击"保存草稿"按钮才能保存。

### Q4: 审批流程是怎样的？

**A**: 当前版本采用固定的两级审批流程：
1. 第一级：部门经理审批
2. 第二级：HR经理审批

只有两级都通过，申请才会最终通过。

### Q5: 可以同时提交多个申请吗？

**A**: 可以。系统没有限制同时提交的申请数量。

### Q6: 审批人可以撤回已经审批的决定吗？

**A**: 不可以。审批决定一旦做出就无法撤回或修改。

### Q7: 如何知道我的申请被谁审批了？

**A**: 在申请详情页面的审批流程部分可以看到所有审批人和他们的审批意见。

### Q8: 申请被拒绝后可以重新提交吗？

**A**: 被拒绝的申请无法重新提交。需要创建一个新的申请，并根据拒绝原因进行调整。

### Q9: 系统支持附件上传吗？

**A**: 当前版本暂不支持附件上传。该功能计划在后续版本中实现。

### Q10: 可以查看其他人的申请吗？

**A**: 
- **员工**: 只能查看自己的申请
- **审批人**: 可以查看需要自己审批的申请
- **管理员**: 可以查看所有申请

## 操作技巧

### 1. 快速填写申请

建议提前准备好以下信息：
- 请假/报销的准确日期
- 详细的原因/用途说明
- 相关金额（报销申请）

### 2. 及时处理审批

审批人应该：
- 定期检查待审批任务
- 及时处理审批请求
- 提供明确的审批意见

### 3. 合理安排时间

提交申请时应该：
- 提前提交，预留审批时间
- 避免在节假日前匆忙提交
- 如有紧急情况，可以线下沟通

### 4. 保持沟通

- 如申请被拒绝，及时与审批人沟通
- 了解拒绝原因，调整后重新申请
- 遇到问题联系系统管理员
